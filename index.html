Quadratic equations

Explanation

Quadratic equations are similar to line equations, but with a number in front that is in the power of 2. For example, while 6x + 4 is a line equation, x2 + 6x + 4 is a quadratic equation. One way we can make it easier, is writing it like ax2 + bx + c, for example, in x2 + 6x + 4, A = 1, B = 6 and C = 4. In 2x2 + 5x + 8, A = 2, B = 5 and C = 4.

Method 1: Factorisation

The first, and the simplest solution is factorising. Basically, when you have a quadratic equation like x2 + 5x + 6, you can see what two numbers multiply to 6, and add up to 5. In this case, it is 2 and 3 as 2 x 3 = 6 and 2 + 3 = 5. However, the problem with this method is that one you have an equation like 2x2 + 5x + 8, you cannot factorise it properly.
